Lista de descripciones

W3C
El W3C es una organización internacional que establece estándares y guías para el desarrollo web. Su objetivo es asegurar la interoperabilidad y accesibilidad de los sitios web, promoviendo la adopción de tecnologías y prácticas que mejoren la experiencia del usuario.

WHATWG
Es una comunidad que desarrolla HTML y APIs para aplicacines web.

HTML
Es un lenguaje de marcas que permite dar estructura y contenido a una página web.

XHTML
Es un lenguaje que adapta HTML para que sea compatible con XML.

DTD
'Document Type Definition'; es un documento que describe cuál es la estructura válida que puede adoptar un archivo XML.

API
'Application Programming Interfaces'; una API es un conjunto de protocolos que permiten a un conjunto de aplicaciones trabajar conjuntamente.

  • Drag and drop
    • Es una funcionalidad que permite arrastrar elementos de una ubicación a otra dentro de una interfaz.

  • Geolocalización
    • Es una funcionalidad que permite al usuario compartir su ubicación geográfica con las aplicaciones que esté utilizando.

  • Almacenamiento local

  • App cache

  • Web workers
    • Son procesos que se ejecutan en segundo plano, al margen de la secuencia de órdenes principal que se esté ejecutando.

WYSIWYG
Un editor WYSIWYG es una aplicación de software que contiene una interfaz que permite a los usuarios ver el resultado final del contenido mientras lo editan. Estas aplicaciones permiten editar contenidos visualmente sin codificar.

Navegadores
Es un tipo de aplicación que permite al usuario acceder a páginas web y navegar por internet.

  • Mozilla Firefox
    • Es un navegador de código abierto y multiplataforma que consume relativamente pocos recursos del ordenador para conectarse a internet.

  • Google Chrome
    • Es el navegador (multiplataforma) de código propietario de Google. Actualmente es el de uso más extendido.

  • Safari
    • Es el navegador (multiplataforma) de código propietario de Apple, incluido en todos los dispositivos informáticos y móviles de la empresa.

  • Opera
    • Es un navegador web multiplataforma que se encuentra entre los más utilizados, con más de cien millones de usuarios.
